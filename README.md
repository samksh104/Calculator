# 🧮 Mini Calculator in C

A simple, menu-driven command-line calculator built using C.  
This project performs basic arithmetic operations: addition, subtraction, multiplication, and division. Great for beginners learning C and working with functions and conditionals.

---

## 🚀 Features
- Addition, Subtraction, Multiplication, Division
- User-friendly CLI menu
- Error handling (e.g., division by zero)
- Simple code structure for learning and extension

---

## 💡 How It Works
- The user selects an operation from a menu
- Inputs two numbers
- The program performs the operation and displays the result
- Loops until the user chooses to exit
